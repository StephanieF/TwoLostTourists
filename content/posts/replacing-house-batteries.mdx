Isolate the inverter and disconnect shore power before touching the battery bay. Confirm
zero volts across the main disconnect with a meter before removing any terminal.

## Pull the old bank

Loosen the negative terminal first, then the positive. Tag each cable as it comes off —
reversed polarity on reassembly will cook the inverter. Lift batteries straight up and out;
AGM units in this bay run heavy, plan for a two-person lift.

## Size the replacement

Match amp-hour capacity to the original bank unless the house load has grown. Confirm bay
dimensions before ordering — a taller case will foul the compartment lid.

| Fastener          | Torque spec     |
| ----------------- | ---------------- |
| Terminal lug bolt  | 90–110 in-lb     |
| Hold-down bracket  | 45–55 in-lb      |
| Bus bar connection | 100–120 in-lb    |

## Reassemble

Set batteries in the tray, hold-downs snug but not overtightened — cracked cases are a
common result of over-torquing the bracket. Connect positive first, negative last. Recheck
every terminal with the meter before restoring shore power.
